From unboxing to fleet member in 30 minutes

Steps
›
1. Hardware
Pi 5 (8GB recommended). 32GB+ microSD. USB-C 5V/5A power supply. Ethernet cable. Optional: Hailo-8 M.2, NVMe SSD.
›
2. Flash
Download Raspberry Pi OS Lite 64-bit. Flash with Raspberry Pi Imager. Set hostname, enable SSH, configure WiFi in imager.
›
3. First Boot
Plug in Ethernet + power. Wait 60 seconds. Find IP on router or 'ping hostname.local'. SSH in: ssh pi@hostname.
›
4. System Update
sudo apt update && sudo apt upgrade -y. Install essentials: curl, git, docker.io, wireguard-tools.
›
5. Join Fleet
Run the BlackRoad fleet join script. Configures WireGuard peer, installs Ollama, sets up Docker, registers with Headscale.
›
6. Verify
ping 10.0.0.1 (Alice). ollama run phi. docker ps. All three work = you're in the fleet.
›
7. Harden
Disable password SSH. Set up fail2ban. Configure firewall (ufw). Enable unattended-upgrades. Done.
